Conceptual Model



Entities & Attributes


User

  • userId (primary key)
  • userHash (user password)
  • userLocation
  • userEmail
  • userPhoneNumber

Post

  • postId (primary key)
  • postStatus
  • postManageStateButtons (delete, display, renew, etc)
  • postTitle
  • postAreaAndCategory
  • postDate
  • postExpiration

Relations

  • One User can have many Posts (1 to n)
  • Each Post can be from only one User
